Hipóteses assumidas:

- O plano cartesiano é o usual, isto é, x na horizontal e y na vertical. 
Assim, a transferência de cor entre os vizinhos é feita da seguinte forma:
Cx > 0 transfere para vizinho da direita; Cx < 0 transfere para o da esquerda.
Cy > 0 transfere para vizinho de cima; Cy < 0 para o de baixo.
Outra consequência é que Cx = C * sen(angle) e Cy = C * cos(angle) 

- A transferência é feita de acordo com a equação
((1 - Cor do vizinho) * Cor do Componente atual)/4
